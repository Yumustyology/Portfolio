Multiverse of Madness is an interactive fantasy-adventure website built with Next.js and Framer Motion. The site showcases fluid, animated scene transitions, character-driven micro-interactions, and responsive layouts optimized for desktop and mobile. The repository contains the full Next.js source and a live demo on Netlify.

Yung Chrome ADS Blocker is a lightweight Chrome extension that blocks unwanted ads across websites. It uses a content script to handle YouTube ad flows, and Manifest V3 declarativeNetRequest rules to target Google Ads and DoubleClick, providing a fast and privacy-friendly browsing experience.

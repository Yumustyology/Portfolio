Hoobank is a polished, futuristic front-end prototype for a fintech application that demonstrates advanced layouts, motion-rich transitions, and responsive UX patterns intended to showcase frontend engineering and design concepts.

The project emphasizes a motion-first UI powered by Framer Motion, component-driven layouts built with Next.js and Tailwind CSS, and type-safe TypeScript components for easier reuse and testing.
